# Command For Run Project

1-clone project

2.copy.env files base .env.example and enter dataabase info

3.git checkout release/v-1

4.composer install

5.npm install

6.php artisan key:generate

7.php artisan migrate

8.php artisan db:seed

9.php artisan serve

10.npm run dev
